# COREPLUGIN

A Minecraft 1.8.8 Bukkit Plugin. 

You have been sentenced to exile on HD 31174 for your crimes against the Emperor. It is a harsh, unforgiving world. But
hope remains. If you could somehow find a way to signal your friends, they might be able to get you offworld somehow...

### Features:

- An overhauled survival experience, focusing on mastering the unforgiving desert
- Custom items to help you on your journey

### Setup:

1. Get a Minecraft 1.8.8 
2. Download + Build CorePlugin
3. Unzip game_end.tar.gz into your server folder
4. Run the server

Claude should be able to help with any of this.
